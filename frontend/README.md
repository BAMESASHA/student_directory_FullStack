# 🎓 StudentHub – Student Directory App

**Student Name:** [Your Name Here]  
**Module:** INFS 202 – Midterm Project

## Description

A React web application for managing and browsing student profiles.
Users can view the directory, search and filter students, view
detailed profiles, and register new students.

## Technologies Used

- React 18
- React Router DOM v6
- Vite (build tool)
- CSS (custom, responsive)

## How to Run

1. Install dependencies:

```bash
   npm install
```

2. Start the development server:

```bash
   npm run dev
```

3. Open in browser: `http://localhost:5173`

## Routes

- `/home` – Welcome page
- `/list` – Student directory with search & filter
- `/details/:id` – Individual student profile
- `/add` – Register a new student
